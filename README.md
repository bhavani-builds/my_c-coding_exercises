This repository contains a collection of beginner-friendly C programs that cover basic concepts such as input/output, arithmetic operations, conditionals, and logic building.
These programs are solved and tested for clarity and understanding.

Programs Included

1. Sum of 5 Subjects
Takes marks of 5 subjects
Calculates total and percentage

3. Reverse of a Number
Reverses any integer without using arrays

3. Currency Note Calculator
Calculates number of ₹100, ₹50, ₹10 notes
Uses division and modulo operators

4. Area & Perimeter of Rectangle
Takes length and breadth
Prints area and perimeter

5. Area & Circumference of Circle
Takes radius
Prints area and circumference

6. Literacy Problem
Calculates literate men and women in a town
Uses percentage formula
